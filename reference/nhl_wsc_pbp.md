# **NHL WSC Play-by-Play**

Returns the narrative-format (Web Services Content) play-by-play feed
for a given NHL game ID from the NHL web service.

This endpoint is distinct from
[`nhl_game_pbp()`](https://fastRhockey.sportsdataverse.org/reference/nhl_game_pbp.md),
which hits the `gamecenter/{id}/play-by-play` endpoint and returns a
tabular play-event feed. `nhl_wsc_pbp()` hits
`wsc/play-by-play/{gameId}` and returns the narrative-format payload
used by NHL.com's story / recap pages (closer in shape to
[`nhl_game_story()`](https://fastRhockey.sportsdataverse.org/reference/nhl_game_story.md)).

## Usage

``` r
nhl_wsc_pbp(game_id)
```

## Arguments

- game_id:

  Integer or character game ID (e.g., 2023020001).

## Value

Returns a list with narrative-format play-by-play data.
