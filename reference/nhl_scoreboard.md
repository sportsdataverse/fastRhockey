# **NHL Scoreboard**

Returns the current league-wide scoreboard with today's games, or — when
`date` is supplied — the scoreboard for a specific date.

## Usage

``` r
nhl_scoreboard(date = NULL)
```

## Arguments

- date:

  Optional character string in `"YYYY-MM-DD"` format. When `NULL`
  (default) the function hits `scoreboard/now`; when a date is provided
  it hits `scoreboard/{date}`.

## Value

A named list of data frames: `gamesByDate`.

**gamesByDate**

|          |           |                                             |
|----------|-----------|---------------------------------------------|
| col_name | types     | description                                 |
| date     | character | Date for which the games are listed.        |
| games    | list      | Nested list of games scheduled on the date. |
