# **Most Recent NHL Season for NHL API**

Returns the most recent NHL season in the format required by the NHL API
(e.g., "20242025"). The NHL season typically runs from October to June,
so a game played in November 2024 belongs to the 2025 season, which is
represented as "20242025" in the NHL API.

## Usage

``` r
most_recent_nhl_season_api_param()
```

## Value

Value for most recent NHL season in the format of the NHL API
