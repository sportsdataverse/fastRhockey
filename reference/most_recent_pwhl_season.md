# **Most Recent PWHL Season**

Returns the concluding year for the most recent PWHL season. The PWHL
season typically runs from January to May, so a game played in March
2025 belongs to the 2025 season.

## Usage

``` r
most_recent_pwhl_season()
```

## Value

Numeric value for the most recent PWHL season year (e.g., 2025).
