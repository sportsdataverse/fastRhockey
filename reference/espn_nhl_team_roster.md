# **Get ESPN NHL Team Roster**

**Get ESPN NHL Team Roster**

## Usage

``` r
espn_nhl_team_roster(team_id, ...)
```

## Arguments

- team_id:

  ESPN team identifier (character or numeric, e.g. `"4"` for Pittsburgh
  Penguins).

- ...:

  Reserved for forward compatibility.
