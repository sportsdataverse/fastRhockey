# **AHL Game Shifts**

All player shift stints for an AHL game (one row per stint).

## Usage

``` r
ahl_game_shifts(game_id)
```

## Arguments

- game_id:

  Numeric AHL game identifier.

## Value

A `fastRhockey_data` data frame, one row per shift stint.
