# **QMJHL Game Corsi/Fenwick (player-level on-ice)**

Player-level on-ice Corsi and Fenwick for a QMJHL game.

## Usage

``` r
qmjhl_game_corsi(game_id)
```

## Arguments

- game_id:

  Numeric QMJHL game identifier.

## Value

A `fastRhockey_data` data frame, one row per player.
