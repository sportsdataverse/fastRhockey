# **QMJHL Game Shifts**

All player shift stints for a QMJHL game (one row per stint).

## Usage

``` r
qmjhl_game_shifts(game_id)
```

## Arguments

- game_id:

  Numeric QMJHL game identifier.

## Value

A `fastRhockey_data` data frame, one row per shift stint.
