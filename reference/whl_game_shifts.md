# **WHL Game Shifts**

All player shift stints for a WHL game (one row per stint).

## Usage

``` r
whl_game_shifts(game_id)
```

## Arguments

- game_id:

  Numeric WHL game identifier.

## Value

A `fastRhockey_data` data frame, one row per shift stint.
