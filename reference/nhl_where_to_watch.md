# **NHL Where to Watch**

Returns streaming/broadcast availability information.

## Usage

``` r
nhl_where_to_watch()
```

## Value

Returns a list with streaming availability data.

## Examples

``` r
# \donttest{
try(nhl_where_to_watch())
#> 2026-06-12 14:20:41.605676: Error fetching where to watch: The API returned an error
#> NULL
# }
```
