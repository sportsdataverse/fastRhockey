# **NHL Playoff Carousel**

Returns playoff series carousel data for a given season.

## Usage

``` r
nhl_playoff_carousel(season = NULL)
```

## Arguments

- season:

  Integer 4-digit year (e.g., 2024 for the 2024-25 season). If NULL,
  uses most recent season.

## Value

Returns a list with playoff series carousel data.
