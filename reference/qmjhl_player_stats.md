# **QMJHL Player Stats**

QMJHL player season stats across all seasons.

## Usage

``` r
qmjhl_player_stats(player_id)
```

## Arguments

- player_id:

  Numeric or character QMJHL player identifier.

## Value

A `fastRhockey_data` data frame, one row per season-stat entry.
